# 🎮 CoinClash

Coin Clash es un juego hypercasual arcade con progresión tipo Idle/Clicker.
Recolecta monedas, mejora tu personaje y desbloquea nuevas zonas.
Optimizado para monetización mediante ads y microtransacciones.
